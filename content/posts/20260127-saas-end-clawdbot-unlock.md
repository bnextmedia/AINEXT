## 這些人是誰？為什麼值得聽

More or Less 是矽谷核心圈的科技 Podcast，四位主持人橫跨創投、大型科技公司和媒體：

**Dave Morin** 是 Offline Ventures 共同創辦人，該基金由 Apple 作為錨定投資人，首期募集 1 億美元。他是 Facebook 早期員工，共同打造了 Facebook Platform 和 Facebook Connect 這兩個改變網路生態的產品，後來創辦了社群應用 Path。Dave 最近幾週密集使用 ClawdBot，成為矽谷最早一批深度體驗「電腦使用」（computer use）能力的創投人。

**Sam Lessin** 是 Slow Ventures 共同創辦人暨普通合夥人，管理近 10 億美元資本，兩度入選 Forbes Midas 種子投資人 25 強。他在 Facebook 擔任過四年產品副總裁，以犀利的產業分析和對 AI 炒作的懷疑態度聞名。

**Brit Morin** 是 Offline Ventures 共同創辦人，曾在 Apple 和 Google 任職，後創辦 Brit+Co 並獲得 4500 萬美元創投資金，現為 Life360（NASDAQ: LIFE）董事。

這三位主持人不只是旁觀者——他們自己就在用 AI 工具建東西、投資 AI 公司、親身經歷商業模式被顛覆的過程。

## 一支 AirPod 的運算能力超過整個阿波羅任務

Dave Morin 在 Offline Ventures 最早期的募資簡報裡就談過一個概念：我們正處在運算的寒武紀時刻。你口袋裡的手機、耳朵裡的 AirPods、手腕上的手錶，加起來的運算能力遠超人類曾經用來登月的全部電腦。光是一支 AirPod 就比整個阿波羅任務的運算能力更強。

但在這集節目裡，Dave 把這個舊觀點推向了一個新結論——而且不是你可能預期的那個。他的結論不是「我們需要更多硬體」，也不是 Apple 傳言中的 AI 穿戴筆有多重要。恰恰相反，他說：「我們已經組裝了一支運算的交響樂團，但還沒有人寫出交響曲。」

改變他想法的，是過去幾週密集使用 ClawdBot 的經驗。ClawdBot 讓他意識到，你現在可以反向工程身邊幾乎任何有運算能力的裝置——牆上的 Samsung Frame 電視、展示照片的數位相框、客廳的 Tonal 健身器材、甚至那些年代久遠的舊電腦。更驚人的是，你可以在幾小時內把這些裝置的 API 串接成一個全新的介面，做出以前完全不存在的體驗。Dave 說他整個週末都在做這種事，而且樂在其中。

他的兒子 Ansel 的故事更具說服力。這個孩子受 Claude Code 啟發，自己用 AI 寫 8-bit 遊戲，存到附 SD 卡的卡匣上，裝進 Palmer Luckey 設計的復刻 Game Boy 裡玩。以前這需要一整個開發團隊，現在一個小孩加上一個 AI 工具就夠了。這不是未來的願景，這是正在發生的事。

## 從訂閱制到「買斷你的資產」

Sam Lessin 聽完 Dave 的描述後，把話題推向了一個更激進的方向。他的出發點是過去十幾年科技業最成功的商業模式——SaaS（軟體即服務）：你不擁有任何東西，每月付費租用一組功能，公司隨時可以改變服務內容、停止支援、調整價格。這個模式的核心是「鎖定」——你被綁在一個生態系裡，離開的成本太高。

但如果 ClawdBot 代表的未來成真，那麼整個「鎖定」的概念就失效了。一切都可以被腳本化、被反向工程、被串接。沒有哪個私有 API 是真正安全的，沒有哪道軟體壁壘是真正堅固的。Sam 認為，這會把商業模式推向一個根本性的逆轉：從「租用能力」回歸到「擁有資產」。而且不只是回到傳統「買斷軟體」（shrink-wrapped software）的時代——而是更進一步，進入一個「什麼都不是封閉原始碼」的世界。你買的不再是一個被限定用途的產品，而是一個可以被任意改造的資源。

Sam 用 John Deere 的例子來說明這種轉變的力道。過去幾年，John Deere 把拖拉機的軟體鎖死，農民只能回原廠維修，即使自己有能力修也不行。這是「租用能力」模式的極端表現，也曾引發大量民怨。但如果本地 AI 代理能破解任何軟體鎖定，農民可以自己逆向工程維修軟體，John Deere 的整個服務收入模式就會從根基崩塌。這不是假設，而是正在逐步發生的事。

## 誰最危險？誰能存活？

那麼，具體來說，哪些公司會受到最大衝擊？節目中三個人各自從不同角度點名了幾個產業。

Sam 首先鎖定的是「硬體加訂閱」模式的公司。他以 Whoop 健身手環和 Eight Sleep 智慧床墊為例，指出這些公司之所以要求消費者每月付高額訂閱費，純粹是因為單靠硬體銷售根本撐不起生意。「我為什麼要為我的床付月租？」Sam 直言不諱。答案是：因為公司沒辦法靠一次性銷售活下去。但一旦消費者能自己建立控制介面，繞過原廠軟體，這些訂閱費的存在理由就消失了。

Dave 從另一個角度補充：那些靠「鬆散的 API 鎖定」維生的軟體公司同樣脆弱。過去，逆向工程一個 API 需要工程師花好幾天甚至好幾週。現在，ClawdBot 可以在幾小時內搞定。所謂的「黏著度」和「轉換成本」，不過是一道暫時的技術屏障，而且這道屏障每個月都在變薄。

Brit 則拿 Peloton 開刀——一台健身腳踏車，每月收 50 美元訂閱費，主要賣的是教練帶領的課程內容。但如果你能破解螢幕、播放任何內容，而且這些內容可以由 AI 根據你的偏好即時生成，Peloton 的訂閱價值就只剩下品牌認同，而品牌認同撐不起每月 50 美元。

不過，節目也討論了可能存活的路徑。Dave 認為 Apple 的 iPhone 本質上已經是一種訂閱——你每隔兩三年升級一次，但 Sam 精準地拆解了消費者真正在買的東西：電池（因為會衰退，你被迫升級）和相機（因為持續進步，你主動想升級）。其他功能根本不是升級的理由。這意味著只有那些硬體本身持續進化、讓消費者覺得「值得掏錢買新一代」的公司，才能在去鎖定的世界裡維持類似訂閱的收入節奏。

Tesla 的 Autopilot 則是另一個有趣的案例。Dave 提到 Tesla 剛宣布把自動駕駛從一次性購買（約 10,000 美元）改為每月 99 美元的訂閱。這看似和去鎖定趨勢背道而馳，但 Sam 認為自動駕駛賣的本質是「降低死亡風險的保險」，而安全這件事很難被消費者自己用 AI 代理「hack 掉」——你不會冒著生命危險去逆向工程你的煞車系統。這是少數訂閱模式依然成立的價值主張，因為它的核心是持續的安全研究更新，而不是一套可以被取代的軟體功能。

## 我的觀察

Dave Morin「運算交響樂團」的比喻非常精準。我自己用 Claude Code 串接各種工具的經驗也印證了這一點——當你能用自然語言控制電腦做任何事，所有的「中間商」都變得脆弱。以前需要工程師花一週寫的整合腳本，現在一句話就能產出。這種能力的普及化，確實會侵蝕所有靠「你做不到」作為護城河的商業模式。

但我認為節目忽略了一個關鍵反論：**大多數人不想自己組交響曲，他們想要的是別人組好、自己按 Play。**

ClawdBot 目前的使用門檻依然很高。Dave 自己在節目裡也承認這一點——他說設定起來「相當技術性」，而且「我不完全確定該怎麼描述這種感覺」。如果連矽谷頂級創投人都覺得難以向別人解釋這東西到底是什麼，那距離普通消費者把它融入日常生活，還有非常長的路要走。技術上的可能性和使用者的實際意願之間，存在一個巨大的鴻溝。

所以，真正的轉變可能不是節目描繪的那種「人人都成為駭客」的場景，而是一個新的中間層崛起——**「新型整合者」取代舊型 SaaS 公司**。這些整合者不賣被鎖定的功能，而是賣「組好的交響曲」：替你串接好所有 API、打造好個人化介面、持續更新維護。他們的護城河不再是技術鎖定，而是體驗品質和持續服務的承諾。

換句話說，SaaS 不會消失，但它的價值基礎會發生根本性的位移——從「你做不到，所以付我錢」轉變為「你做得到但不想花時間，所以付我錢」。前者靠壁壘，後者靠品質。這是一個巨大的差異，對消費者來說毫無疑問是好消息。
