## 專業分工的黃昏

自工業革命以來，專業分工一直是現代經濟的核心邏輯。工廠裡每個人負責產品的一小部分，組裝線上每個崗位都高度專門化。這種模式有其道理：讓每個人盡可能專精、盡可能可替換，整體效率就能最大化。但 Replit 執行長 Amjad Masad 認為，這個延續了兩百多年的模式即將終結。

原因很簡單。當你的 HR 專員能用 AI Agent 在三天內做出一套組織圖軟體，她同時也是軟體工程師了。當她能快速學會任何新技能——因為有 Agent 幫忙——她也可能成為行銷人員、資料分析師、專案經理。AI Agent 的出現，讓「一人多職」從口號變成現實。

Masad 觀察到，工作將變得更不專門化、更不封閉。過去公司有行銷部門、業務部門、工程部門，各自為政。未來的組織架構會從階層式變成網絡式，更像開源專案而不是傳統企業。每個員工早上醒來，任務不再是「寫這封行銷信」或「優化這個按鈕」，而是「讓公司成功」。這聽起來很抽象，但 Masad 的意思是，每個人的職責將從「完成被指派的任務」轉變為「創造價值」。

Replit 自己已經開始這樣實驗了。他們第一次建立真正的產品團隊，但這個團隊的組成方式很特別：設計師、工程師、產品經理的角色，經常由同一個人擔任。Masad 說他們正在試著把許多傳統上分開的角色合併在一起，打造所謂的「通才員工」。

## 文科教育會變得更有價值嗎？

在 Q&A 環節，有人問了一個有趣的問題：既然清晰思考和想法在未來如此重要，這是否意味著博雅教育、批判性思維訓練，會比 STEM 技能更有價值？

Masad 的回答很務實。他認為這兩者並不互斥，但文科確實會變得更有價值。他觀察到，今天的工程師往往過於狹隘，比他們在未來能承受的還要狹隘。很多工程師甚至不了解自己所在的業務，只專注於非常窄的技術領域。這在過去或許還行得通，但當每個人都必須成為通才時，這種狹隘就會成為致命傷。

未來的人需要更廣闊的世界觀和技能組合。你不能只懂程式、不懂商業；不能只懂技術、不懂人性。但 Masad 也強調，科學思維依然重要。這不是「文科取代理科」的故事，而是「兩者都需要，而且需要整合」的故事。

## 給年輕人的具體建議

有人問 Masad：如果通才員工是未來趨勢，現在應該找什麼樣的工作來培養這種能力？不是每個人都想馬上創業，有些人想先累積經驗、學習團隊合作。

Masad 的答案非常直接：盡早加入新創公司。

他說，你可以把它想成一條指數衰減曲線。創辦人能獲得最多的通才經驗，第一號員工次之，到了第一百號員工時，你能獲得的通才經驗就大幅下降了。所以要盡可能早加入，視你能承受的風險程度而定。就算是加入一家 B 輪公司的第二十號員工，你獲得的經驗也會比進入 FAANG 大公司多很多。

但光是加入新創還不夠。Masad 強調，就算你進了新創，你還是需要主動尋找通才機會。不要坐在那裡等別人給你任務。他說自己的心態是這樣的：早上醒來，不是看待辦清單，而是看使命。我的使命是讓這家公司成功、讓它更有價值。

這是一個微妙但關鍵的心態轉換。「完成待辦清單」是螺絲釘的思維；「讓公司成功」是創業者的思維。在 AI Agent 時代，螺絲釘會被取代，創業者不會。

## 領域知識依然是王道

另一個值得注意的觀點出現在討論 Agent 新創的環節。有人問，Agent 領域是否已經過度擁擠？Masad 承認，軟體工程 Agent 確實競爭激烈，如果你現在才進場，必須有真正新穎的想法才能勝出。但他指出，還有很多領域沒人在做：HR Agent、財務 Agent、合規 Agent。

他給的建議是：從你自己感興趣、有領域知識的地方開始。打造 Agent 公司最重要的不是技術能力，而是領域專業。如果你自己就是合規專員，或者對合規充滿熱情，那你就應該去做合規 Agent，因為你會學得最快、也擁有最深的領域知識。

這呼應了他對通才的定義。通才不是什麼都懂一點、什麼都不精的人。通才是有能力跨越多個領域、把不同知識整合起來解決問題的人。你還是需要在某些領域有深厚的根基，只是這個根基的邊界會變得更模糊、更流動。

## 我的觀察

Amjad Masad 描述的「通才員工」願景，其實是對我們這一代人被灌輸的職涯觀念的正面挑戰。我們從小被教育要成為「專家」——選一個領域、深耕十年、成為該領域的權威。大學選系、研究所選組、工作選職能，每一步都在收窄你的路徑。這套邏輯在工業時代是合理的，因為專業分工確實能提高效率。

但 Masad 描述的未來是：有想法的通才，比只會執行的專家更有價值。當 Agent 能幫你快速學會任何技能、快速執行任何任務，你的稀缺性就不再來自「你會做什麼」，而是「你知道該做什麼」。知道該做什麼，需要的是廣闊的視野、對商業的理解、對人性的洞察、對問題的敏感度。這些東西很難被 Agent 取代，因為它們需要跨領域的整合能力和對真實世界的直覺。

這對台灣的職場文化是一個巨大挑戰。我們的企業普遍還在用工業時代的 KPI 管理員工——你的 OKR 是什麼、這季達成率多少、你負責的那一小塊做得怎樣。這種管理方式假設每個人是一顆螺絲釘，可以被精確測量、可以被輕易替換。但 Masad 說的「網絡型組織」完全不是這樣運作的。在網絡型組織裡，你的價值不是來自你完成了多少被指派的任務，而是你為整體創造了多少新價值。這需要完全不同的績效觀念、完全不同的管理文化。

更深層的問題是：我們的教育體系準備好培養通才了嗎？台灣的升學制度依然高度分流，高中就要選組、大學就要選系，跨領域學習的成本很高。Masad 說文科和理科不互斥、兩者都需要，但我們的系統卻強迫學生在很早的階段就做出非此即彼的選擇。這種結構性的限制，比任何一堂課程的內容都更難改變。

Masad 給的職涯建議——盡早加入新創、主動尋找跨領域機會、用創業者心態而非螺絲釘心態工作——聽起來簡單，執行起來很難。因為這不只是找工作的策略，而是對「什麼是好工作」的根本重新定義。好工作不再是穩定、專業、可預期的工作；好工作是讓你有機會變成通才的工作。這需要的不只是個人的勇氣，也需要整個社會對「職涯成功」的想像進行一次大規模的軟體更新。
